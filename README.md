## BookingApp - мобильное приложение для бронирования номеров отелей
## Главный экран
Данный экран отображает следующую информацию:
- фотографии отеля
- основные данные: рейтинг, название, адрес, стартовая цена
- подробные данные об отеле: особенности отеля, описание, опции
- кнопка выбора номера

## Экран "Номер"
Данный экран отображает следующую информацию:
- название отеля в навигационной панели
- список номеров отеля
- каждый номер содержит следующие данные: фотографии номера, название, особенности, цена за количество дней отдыха
- кнопка бронирования номера

## Экран "Бронирование"
Данный экран отображает следующую информацию:
- блок с данными отеля
- блок с данными бронирования (страна. даты и др.)
- данные о покупателе (номер, почта)
- информация о туристах и их количестве
- информация о итоговой цене
- кнопка "Оплатить"
- учтена обработка ошибок ввода полей

## Экран "Оплачено"
Открывается в случае успешного бронировании номера отеля.
Данный экран отображает следующее:
- сообщение с информацией о номере заказа и времени доставки
- кнопка, возвращающая на начальный экран
